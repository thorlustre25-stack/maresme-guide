# Maresme Tourism Website Project

## Project Vision
A tourism website targeting budget-conscious tourists visiting the Maresme coast of Catalunya, focusing on beaches, nightlife, food, and practical travel information.

## Target Audience
- Beach-focused tourists (summer season)
- Party/nightlife seekers
- Families visiting Barcelona by train
- Budget travelers looking for practical info

## Core Features

### 1. Practical Information
- **Airport Bus Routes**: Clear, step-by-step directions to bus stops
- **Train to Barcelona**: Schedules, stations, ticket info
- **Tourist Information Offices**: Locations and hours

### 2. Beaches & Xiringuitos
- Beach listings with facilities info
- Xiringuito (beach bar) directory with:
  - Location
  - Opening hours
  - Events/parties
  - Contact info

### 3. Food & Nightlife
- Restaurant listings
- Bar directory
- Party/event calendar
- Featured establishments (paid)

### 4. Excursions
- Sea activities
- Day trips
- Beach activities
